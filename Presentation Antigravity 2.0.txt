Slide 2 — What are AI Models?
AI models are trained systems that learn patterns from large amounts of data.
Purpose

Predict outputs
Understand inputs
Generate responses
Perform intelligent tasks
Types of AI Models
Text, Image, Audio, Video, Multimodal models
Examples
GPT · Gemini · Claude · Llama

Slide 3 — What are LLMs?
LLM = Large Language Model
Advanced AI models trained on internet-scale text datasets to understand, analyze, and generate human-like language using deep neural networks (Transformers).
How they work
They process input text and calculate the most probable next word (token) in a sequence.
Core Capabilities
Answer complex questions & Generate functional code
Summarize lengthy documents & Translate between languages
Assist in brainstorming, problem-solving, and logic analysis
Popular LLMs
GPT-4o · Claude 3.5 · Gemini 1.5 · DeepSeek-V3 · Llama 3
Limitation
LLMs mainly act as chatbots. They respond to prompts but cannot independently execute long-running, multi-step workflows or manage tools without an external orchestrator.

Slide 4 — What are AI Agents?
AI Agents are intelligent systems that can understand goals, make decisions, use tools, and execute tasks autonomously.
The Agent Paradigm
By combining an LLM with memory, reasoning loops, and tool access, agents break free from the chatbox to perform real-world tasks.
Unlike Traditional LLMs, Agents Can:
Perform actions
Use external systems
Coordinate workflows
Operate with memory and context
Examples
Autonomous coding agents · Research agents · Workflow automation agents · AI copilots

Slide 10 — Parallel Subagents
Subagents are smaller AI workers created dynamically for parallel execution.
Parallel Processing: By spawning specialized subagents (e.g., a researcher, a coder, a reviewer), complex problems are solved much faster through concurrent execution.
How They Work
Main agent splits a complex task
Multiple subagents handle different parts
Results are combined automatically
Advantages
Faster processing · Parallel problem solving · Better scalability · Improved task specialization
